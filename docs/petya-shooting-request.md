# Запрос на съёмку для Пети/Влада/менеджера

Этот файл можно переслать целиком — здесь нет технических деталей, только
то, что нужно снять и как прислать.

## Правила съёмки

- **Обычный товар** — 3 фото на товар, строго в таком порядке:
  1. `card-front` — красивая карточка/лицевая сторона товара;
  2. `front` — обычное фото товара спереди;
  3. `back` — обычное фото товара сзади (состав, штрихкод).
- **Парфюм 5 мл** — достаточно одной designed-карточки, фото спереди/сзади
  не нужны.
- Если есть возможность — подписывайте название товара и код 1С прямо в
  сообщении с фото (это сильно ускоряет привязку фото к товару).
- **Не смешивайте разные товары в одной тройке фото.** Если в 1 сообщении
  или альбоме 3 фото — все три должны быть про один и тот же товар/вариант
  (даже если это очень похожие товары одного бренда). Именно перепутанные
  тройки сейчас держат несколько товаров Dove в подвешенном состоянии — см.
  раздел "Незакрытые товары" ниже.

## Как отправлять Пете

- Присылайте фото пачками по **3 штуки на товар**: card-front, front, back.
- Можно присылать сразу по **9 фото = 3 товара**, если снимаете несколько
  товаров подряд — но каждая тройка внутри должна быть одним товаром.
- Для парфюма 5 мл достаточно одной карточки на товар, но название и цену
  для него нужно прислать отдельным текстом (по фото название не всегда
  однозначно понятно).
