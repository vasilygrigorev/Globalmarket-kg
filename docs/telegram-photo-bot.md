# Telegram Photo Bot

Локальный Telegram-бот принимает фото товара, запускает `scripts/local_photo_bot.py` и отправляет обработанные JPG обратно в Telegram.

## Что умеет

- Принимает 1-2 фото товара.
- Принимает выгрузки остатков `.mxl`, `.xls`, `.xlsx`, `.csv` и сохраняет их локально.
- Сохраняет входные фото в `assets/telegram_inbox/`.
- Сохраняет выгрузки остатков в `assets/telegram_uploads/stock/`.
- Обрабатывает фото локально без AI и без токенов.
- Возвращает обработанные файлы в Telegram.
- Кладет готовые изображения в `assets/products/`.

## Запуск

1. Создать бота в Telegram через `@BotFather`.
2. Скопировать token.
3. Запустить локально:

```bash
export TELEGRAM_BOT_TOKEN="123456:token_from_botfather"
python3 scripts/telegram_photo_bot.py
```

Можно изменить долю товара в кадре:

```bash
python3 scripts/telegram_photo_bot.py --object-fill 0.8
```

## Команды в Telegram

```text
/start
```

Показывает краткую подсказку.

```text
/slug clear-men-cr7-600ml
```

Задает имя товара/файлов. После этого можно отправить фото.

```text
/process
```

Принудительно запускает обработку текущей пачки фото.

```text
/clear
```

Очищает текущую пачку фото.

## Как отправлять фото

Лучший вариант:

1. Написать `/slug clear-men-cr7-600ml`.
2. Отправить 1-2 фото товара.
3. Подождать несколько секунд.
4. Бот вернет обработанные JPG.

Если отправить фото с подписью, подпись используется как slug.

## Как отправлять выгрузку 1С

Отправить файл в Telegram как документ. Например:

```text
остатки02062026.mxl
```

Бот сохранит файл в:

```text
assets/telegram_uploads/stock/
```

После обновления кода бота старый запущенный процесс нужно перезапустить, иначе документы будут игнорироваться старой версией.

## Важно

- Бот работает только пока запущен локальный скрипт.
- Фото обрабатываются на этом компьютере.
- Токен не нужно записывать в код или документы.
- Для постоянной работы позже лучше запускать бота на VPS или на отдельном офисном компьютере.
